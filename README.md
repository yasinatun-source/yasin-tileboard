# Yasin TileBoard

This is a custom TileBoard interface for Home Assistant.
It provides a fast, tile-based dashboard alternative.

### Installation (HACS)

1. HACS → Custom Repositories → Add
2. Repository: `https://github.com/YOUR_USERNAME/yasin-tileboard`
3. Category: `Dashboard`
4. Install → Restart Home Assistant

### Access

After installation:
Open → `/local/tileboard/index.html` or `/hacsfiles/yasin-tileboard/tileboard/index.html`
